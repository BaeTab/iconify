# Iconify - 무료 PNG/JPG to ICO 변환기 ⚡

PNG, JPG, WEBP 이미지를 윈도우용 아이콘 파일(.ico)로 즉시 변환해주는 웹 애플리케이션입니다.

![Iconify Preview](https://via.placeholder.com/800x400?text=Iconify+Preview)

## ✨ 주요 기능

*   **100% 클라이언트 처리**: 이미지가 서버로 전송되지 않고 브라우저에서 직접 변환되어 개인정보 유출 걱정이 없습니다.
*   **다양한 해상도 지원**: 하나의 ICO 파일에 16x16, 32x32, 48x48, 128x128, 256x256(Jumbo) 등 다양한 크기의 아이콘을 포함합니다.
*   **고품질 변환**: 크기 조절 시 `high-quality` 스무딩을 적용하여 선명한 결과물을 제공합니다.
*   **간편한 사용**: 드래그 앤 드롭으로 파일을 쉽게 업로드할 수 있습니다.
*   **즉시 미리보기**: 변환될 아이콘의 모습을 사이즈별로 미리 확인할 수 있습니다.

## 🛠️ 기술 스택

*   **Frontend**: React (Vite), TypeScript
*   **Styling**: Tailwind CSS
*   **Deploy**: Firebase Hosting
*   **Etc**: React Helmet Async (SEO), Canvas API (Image Processing)

## 🚀 시작하기

### 설치

```bash
# 의존성 패키지 설치
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드 및 배포

```bash
# 프로덕션 빌드 후 Firebase Hosting 배포
npm run deploy
```
